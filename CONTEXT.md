# Dark Software Factory

This context defines the language for a system that turns high-level software intent into verified, deployable changes through agentic automation.

## Language

**Dark Software Factory**:
An automated software delivery system where agents perform implementation, validation, and deployment work while humans define intent, policies, and audit outcomes.
_Avoid_: Software 3.0, AaaS, agent factory

**VPS Foundation**:
The baseline server environment that gives the factory a repeatable place to run early workloads before larger orchestration or fleet infrastructure exists.
_Avoid_: Server setup, VPS setup, host bootstrap
